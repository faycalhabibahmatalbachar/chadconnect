# XAMPP / MySQL (ChadConnect)

## 1) Démarrer XAMPP
- Lancer **XAMPP Control Panel**
- Start **MySQL**
- (optionnel) Start **Apache** pour phpMyAdmin

## 2) Importer la base
1. Aller sur `http://localhost/phpmyadmin`
2. Onglet **Import**
3. Importer `database/schema.sql`
4. Importer `database/seed.sql`
